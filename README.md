# 文字分享卡片生成器

一个极简的单文件文字分享卡片生成工具，专为移动端优化。

## ✨ 特性

- 📱 **移动端优化** - 专为移动端设计的排版和字号
- 🎨 **8种精美主题** - 白霜、暗夜、暖沙、海蓝、森林、暮色、薰衣草、墨色
- 🔤 **3种字体选择** - 思源宋体、思源黑体、系统默认
- 📐 **3种图片比例** - 4:5、1:1、9:16，适配各种社交平台
- 🔗 **二维码生成** - 输入链接自动生成二维码
- 💾 **一键下载** - 导出高清PNG图片
- 🚀 **单文件部署** - 所有代码在一个HTML文件中，无需构建
- 🎯 **实时预览** - 输入即时更新预览

## 🎨 设计理念

- **极简主义** - 去除多余装饰，专注内容本身
- **优雅排版** - 参考苹果设计风格，精心调整字间距、行高、留白
- **最少打扰** - 简洁的界面，清晰的功能

## 🔧 技术栈

- 纯HTML + CSS + JavaScript
- Canvas API 绘制卡片
- QRious 生成二维码
- Google Fonts (思源宋体、思源黑体)

## 📝 使用说明

1. **输入文字** - 在文本框中输入或粘贴你想分享的内容
2. **选择风格** - 选择喜欢的主题、字体和比例
3. **添加链接**（可选）- 输入链接会自动生成二维码
4. **下载卡片** - 点击"下载 PNG"保存图片
